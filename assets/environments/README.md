## Blender Export Settings
Forward Axis: Z
Up Axis: Y
Objects: Apply Modifiers (true)
Geometry: UV coordinates (true), Normals (true), Triangulated Mesh (true)
Grouping: Object groups (true)

After you export the file, open the .mtl file and ensure that the file path for the texture is just
- texture.png
- NOT ex. /Downloads/texture.png
